# step-by-step-apis-development-in-nodejs-with-sequelize-orm

Step by Step APIs Development in Node JS with Sequelize ORM

- Init Repository

```npm
npm init
```

- Install Express

```npm
npm i express --save
```

- Run App

```npm
node app.js
```

- Install Nodemon

```npm
npm -g nodemon
```

- Execute Nodemon

```npm
nodemon
```

- Execute Nodemon in Specific File

```npm
nodemon server.js
```
